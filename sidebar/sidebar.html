How to Use EV Scorer

Saving Listings

When browsing supported car sites, a purple "Save to EV Scorer" button appears on EV listing pages. Click it to save the listing.

Tip: Use Alt+S to quick-save any page.

Supported Sites

  • AutoTrader.ca
  • Kijiji.ca
  • Clutch.ca
  • CarGurus.ca
  • CanadaDrives.ca

Understanding Scores

Each vehicle is scored 0-100 using Multi-Criteria Decision Analysis (MCDA). Higher scores indicate better value.

70+ Excellent
40-69 Good
<40 Fair

Scoring Criteria

Price35%Lower is better
Odometer16%Lower is better
Range12%Higher is better
Year10%Newer is better
Trim Level10%Higher is better
Distance10%Closer is better
Remote Start10%Fob+App best
Length10%Shorter is better
Damage5%Less is better
Heat Pump5%Yes is better

Keyboard Shortcuts

Alt+E Open popup
Alt+S Quick save

Data & Privacy

All data is stored locally in your browser. Use Export/Import in the popup to backup your listings.
